# URL编码/解码 Alfred 插件

## 功能

- 一键进行URL编码和解码
- 支持多种URL编码格式：
  - 标准URL编码（保留斜杠和部分特殊字符）
  - 查询参数编码（将空格编码为+，适用于查询参数）
  - 完全URL编码（编码所有特殊字符）
- 自动检测并提供解码选项（当输入已编码文本时）

## 使用方法

1. 在 Alfred 中输入 `urlencode` 后跟要编码的文本
   例如：`urlencode Hello World!`
2. Alfred 将显示多种编码格式的结果
3. 选择您需要的格式，按回车键复制到剪贴板
4. 如果输入的是已编码的文本，还会显示解码选项

## 安装

1. 下载最新版本的插件
2. 双击下载的文件安装到 Alfred

## 要求

- Alfred 4 或更高版本
- Alfred Powerpack
- Python 3
